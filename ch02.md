## 2.1 코드 작성 규칙
### 세미콜론 : 문장의 끝을 의미
- 붙여도 되고 안 붙여도 됨
- 가능한 붙이는 것을 권장

### 주석 : comment
- 한 줄 주석
  - Ctrl + /
  - //
- 여러 줄 주석
  - Alt + Shift + A
  - /* ~~~ */

### 들여쓰기 : indent
- tap size
